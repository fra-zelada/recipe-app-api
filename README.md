# recipe-app-api
Recipe api project  - DRF course

## run the project

``` docker-compose up ```

## construir imagen

``` docker build . ```

or

``` docker-compose build ```


## correr linter

``` docker-compose run --rm app sh -c "flake8" ```

## crear el proyecto

``` docker-compose run --rm app sh -c "django-admin startproject app ." ```
